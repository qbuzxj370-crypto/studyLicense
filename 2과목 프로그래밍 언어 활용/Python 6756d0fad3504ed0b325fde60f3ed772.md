### 데이터 처리

**range() 함수**

연속된 정수 시퀀스를 만들어 반복에 사용

실제로는 리스트를 다 만들지 않고 필요할 때 값을 생성하는 지연(lazy) 시퀀스라 메모리 효율적

정수들의 “범위”를 표현하는 range 객체를 반환

형식

- range(start, stop, step)
    - stop (종료값) : 포함하지 않는(half-open) 규칙이 핵심
    - start (초기값) : 생략 시 0이 기본값
    - step (증감값) : 생략 시 1이 기본값, 0은 불가능

**슬라이스 (Slice)**

문자열이나 리스트와 같은 순차형 객체에서 일부를 잘라(Slicing) 반환

형식

- 객체명[초기위치:최종위치]
    - 초기위치에서 최종위치-1까지의 요소들을 반환
- 객체명[초기위치:최종위치:증감값]
    - 초기위치에서 최종위치-1까지 증감값만큼 변경되면서 해당 위치의 요소들을 반환
    - 증감값이 음수인 경우 초기위치에서 최종위치+1까지 증감값만큼 감소하면서 해당 위치의 요소들을 반환
- 객체명[:], 객체명[::]
    - 객체의 모든 요소를 반환
- 객체명[초기위치:]
    - 객체의 초기위치에서 마지막 위치까지의 요소들을 반환
- 객체명[:최종위치]
    - 객체의 0번째 위치에서 최종위치-1까지의 요소들을 반환
- 객체명[::증감값]
    - 객체의 0번째 위치에서 마지막 위치까지 증감값만큼 증가하면서 해당 위치의 요소들을 반환
